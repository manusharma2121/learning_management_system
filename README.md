📚 Mini Learning Management System (LMS)

A simple Python-based console Learning Management System with Teacher and Student Login and Role-Based Menu functionality.

This project demonstrates object-oriented programming (OOP), authentication logic, course management, and progress tracking in Python.

🚀 Features
👨‍🏫 Teacher Features

Login with predefined credentials

Add new courses

View all available courses

View enrolled students in a course

View progress of any registered student

👩‍🎓 Student Features

Login (Auto-register if not found)

View available courses

Enroll in courses

Mark lessons as completed

Track personal progress (percentage based)

🏗️ Project Structure

The system is built using:

Class Course

Stores course title

Number of lessons

List of enrolled students

Class Student

Username and password

Completed lessons tracking (dictionary)

Global Data

courses list

students list

Predefined teachers list
